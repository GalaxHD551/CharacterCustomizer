## Character Customize

A simple UI for customizing your onset caracter.

### Feature
- Clean UI
- Camera zoom on body
- Allow to rotate the camera using the mouse
- Networked cloths
- Easy to modulate
- Allow Custom models
- Can be easy added to a project
